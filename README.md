# ♠ VibePoker

A WPT-inspired Texas Hold'em poker game built with vanilla JavaScript and CSS. Play against three AI opponents with realistic betting strategies and hand evaluation.

## Features

- **Texas Hold'em Gameplay** - Full poker mechanics with pre-flop, flop, turn, and river betting rounds
- **AI Opponents** - Three intelligent AI players with adaptive decision-making based on hand strength and pot odds
- **Hand Evaluation** - Accurate hand ranking from High Card to Royal Flush
- **Smooth UI** - Animated cards, chip displays, and real-time pot updates
- **Interactive Tutorial** - Introduction script to learn the game
- **Game States** - Win/lose/quit screens with stats tracking

## How to Run

1. Open `index.html` in a modern web browser
2. Follow the tutorial or click "Deal Now" to start playing
3. Use the action buttons (Fold, Check, Call, Bet, Raise, All-in) to play your hand

## Game Controls

- **Fold** - Forfeit your hand
- **Check** - Pass without betting (if allowed)
- **Call** - Match the current bet
- **Bet** - Place your own bet
- **Raise** - Increase the current bet
- **All-in** - Bet all remaining chips
